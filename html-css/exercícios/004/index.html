Olha o favicon aqui em cima!!

Para adicionar um favicon em um documento HTML, você pode usar a tag <link> dentro da seção <head>.

A tag <link> é usada para vincular recursos externos, como folhas de estilo e ícones, ao documento HTML. Para adicionar um favicon, você deve especificar o caminho do arquivo de ícone usando o atributo href e definir o tipo de ícone com o atributo rel.

O código para adicionar um favicon é o seguinte:

    <link rel="shortcut icon" href="caminho/para/seu/favicon.ico" type="image/x-icon">

Onde em "caminho/para/seu/favicon.ico" você deve substituir pelo caminho real do arquivo de ícone que deseja usar como favicon.

O atributo type é opcional, mas é recomendado para especificar o tipo MIME do ícone. O valor image/x-icon é comumente usado para arquivos de ícone no formato ICO.
