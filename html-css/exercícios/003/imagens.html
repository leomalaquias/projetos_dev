Adicionando imagens em documento HTML5

Para adicionar imagens em um documento HTML, você pode usar a tag <img>.

A tag <img> é usada para incorporar imagens em uma página web. Ela não possui uma tag de fechamento, pois é uma tag auto-fechante.

Você pode especificar o caminho da imagem usando o atributo src e fornecer um texto alternativo usando o atributo alt.

Para que seja possível selecinonar a imagem com o comando img quando colocamos o comando com o atributo src
Conseguimos o segunte resultado:

    <img src="caminho/para/sua/imagem.jpg" alt="Descrição da imagem">

Onde em "caminho/para/sua/imagem.jpg" entre as aspas, podemos utilizar as teclas ctrl + espaço para selecionar o caminho da imagem que queremos adicionar.
O atributo alt é importante, pois quando a imagem não puder ser exibida ou for realizado uma descrição por questões de acessibilidade será exibido a mensagem contida no atributo alt

Por exemplo, para adicionar uma imagem chamada imagem.jpg localizada na mesma pasta do arquivo HTML, você pode usar o seguinte código:

    <img src="imagem.jpg" alt="Descrição da imagem">

Se a imagem estiver em uma subpasta chamada imagens, você pode especificar o caminho relativo assim:

    <img src="imagens/imagem.jpg" alt="Descrição da imagem">

Você também pode ajustar o tamanho da imagem usando os atributos width e height. Por exemplo:

    <img src="imagens/imagem.jpg" alt="Descrição da imagem" width="300" height="200">

Isso irá redimensionar a imagem para 300 pixels de largura e 200 pixels de altura.

Abaixo estão mostrados os exemplos de cada um dos itens acima


    <img src="imagens/logo_html_full.png" alt="Logo HTML">
  

    <img src="imagens/logo_css_full.png" alt="Logo CSS">
  

    <img src="imagens/logo_html_full.png" alt="Logo HTML" width="300" height="200">

    e

    <img src="imagens/logo_css_full.png" alt="Logo CSS" width="300" height="200">

Sabendo esses atributos citador por último podemos alterá-los para que seja possível obter imagens que estejam adequadar e sem distorção

as imagens podem ser editadas também para o formato desejado antes de serem colocadas no código, por aplicações como GIMP, Canva, Adobe Photoshop e Corel Draw

Outra possibilidade é copiar o endereço de uma inagem da web para que seja adicionada a código, para isso devemos copiar o endereço da imagem que desejamos adicionar ao código (importante procurar uma imagem com o tamanho final desejado ou redimensioná-la utilizando os atributos width e height), esse processo pode ser realizado através do comando:

    <img src="endereço da imagem" alt="Descrição da imagem">

    <img src="https://cdn.iconscout.com/icon/free/png-256/free-javascript-2038874-1720087.png" alt="Logo JavaScript">

Para deixar a mesma altura das outras imagens podemos utilizar os atributos width e height

    <img src="https://cdn.iconscout.com/icon/free/png-256/free-javascript-2038874-1720087.png" alt="Logo JavaScript" width="200" height="200">

A logo do JavaScript não se igualou ao tamanho das outras uma vez que as margens transparentes da imagem são diferentes para as 3 imagens, por isso é importante que sejam editadas as imagens anteriormente ou que sejam alteradas via width e height ou ainda via CSS para que se adequem ao site e seja obtido uma estrutura visual simétrica.
