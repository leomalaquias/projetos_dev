Texto marcado

Podemos criar também um texto marcado usando a tag "mark"

Quando vamo fazer uma alteração pontual podemos ussar CSS (style) junto com a tag original como por exemplo mark, se todas forem seguir o mesmo padrão é necessário colocar no cabeçalho, criando um seleto para a tag desejada, assim vai adicionar em todos os itens continos na marcação
